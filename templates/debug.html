üîç Troubleshooting Guide

If you see empty CSV:

  • Check if API response contains data (look at "Data Found" above)
  • Verify your search parameters (lat/lng, radius, acreage range)
  • Try a different location or larger search radius
  • Check if your RapidAPI subscription is active

If API call fails:

  • Verify your RAPIDAPI_KEY is set correctly in Vercel environment variables
  • Check if you have remaining API credits
  • Ensure your RapidAPI subscription includes the PRYCD Comps API

If authentication fails:

  • Make sure your SECURE_KEY environment variable matches what you're entering
  • Check for extra spaces or special characters
